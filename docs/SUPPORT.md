# Support

Thank you for using our project! We're here to help you. Below you will find information on how to get support.

## Getting Help

### Issues

If you encounter a bug, have a feature request, or need help with something specific, please use our [issue tracker](https://github.com/JSChronicles/emberpost/issues). Make sure to search for existing issues before opening a new one. This helps us avoid duplicates and allows us to provide better assistance.

### Documentation

Before reaching out for help, please review our [README](https://github.com/JSChronicles/emberpost/blob/main/README.md). It may cover some common questions and issues.


## Contributing

We welcome contributions from others! Please read our [Contributing Guidelines](https://github.com/JSChronicles/emberpost/blob/main/docs/CONTRIBUTING.md) to learn how you can help.


Thank you for your support and contributions!
